📜 滚动条组件

基础滚动条

这是一段很长的内容，用来测试滚动条功能。

滚动条组件提供了自定义的滚动条样式，替代了浏览器默认的滚动条。

支持垂直和水平滚动，可以根据内容自动显示或隐藏。

滚动条组件支持多种配置选项，如高度、最大高度、样式类等。

可以通过 always 属性让滚动条始终显示，即使内容不需要滚动。

滚动条组件还提供了丰富的公共方法，如 scrollTo、scrollToTop 等。

支持鼠标拖拽滚动条进行滚动操作，提供更好的用户体验。

滚动条组件完全基于 Web Components 标准，无需额外依赖。

样式可以通过 CSS 变量进行自定义，支持主题切换。

响应式设计，在不同屏幕尺寸下都能正常工作。

支持设置最小滚动条尺寸，确保滚动条始终可见。

滚动条组件与 Element Plus 的 el-scrollbar 功能保持一致。

始终显示滚动条

这个滚动条始终显示，即使内容不需要滚动。

通过 always 属性可以强制显示滚动条。

适用于需要保持布局一致性的场景。

自定义样式

这个滚动条使用了自定义的样式类名。

wrap-class 用于包装器元素的样式。

view-class 用于内容视图的样式。

可以通过这些类名进行样式定制。

支持内联样式和外部样式表。

样式隔离，不会影响其他组件。
