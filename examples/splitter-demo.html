EW Splitter 分割面板组件

基于 Element Plus 设计的分割面板组件，支持拖拽调整大小和折叠功能

基础用法

最简单的分割面板，支持拖拽调整大小

左侧面板 (30%)
右侧面板 (70%)
<ew-splitter> <ew-splitter-pane size="30%"> <div>左侧面板 (30%)</div> </ew-splitter-pane> <ew-splitter-pane size="70%"> <div>右侧面板 (70%)</div> </ew-splitter-pane> </ew-splitter>

可折叠面板

支持折叠功能的面板，鼠标悬浮到分割线上会显示折叠按钮

面板1 (可折叠)
面板2 (可折叠)
面板3 (不可折叠)
面板4 (可折叠)
<ew-splitter> <ew-splitter-pane size="25%" collapsible min="10%"> <div>面板1 (可折叠)</div> </ew-splitter-pane> <ew-splitter-pane size="25%" collapsible> <div>面板2 (可折叠)</div> </ew-splitter-pane> <ew-splitter-pane size="25%"> <div>面板3 (不可折叠)</div> </ew-splitter-pane> <ew-splitter-pane size="25%" collapsible> <div>面板4 (可折叠)</div> </ew-splitter-pane> </ew-splitter>

垂直布局

垂直方向的分割面板

上方面板 (40%)
下方面板 (60%)
<ew-splitter layout="vertical"> <ew-splitter-pane size="40%" collapsible> <div>上方面板 (40%)</div> </ew-splitter-pane> <ew-splitter-pane size="60%" collapsible> <div>下方面板 (60%)</div> </ew-splitter-pane> </ew-splitter>

功能特性

  • 支持水平和垂直布局
  • 拖拽调整面板大小
  • 面板折叠/展开功能
  • 最小/最大尺寸限制
  • 可禁用拖拽调整
  • 响应式设计
  • 暗色主题支持
  • 完全兼容 Element Plus API

API 参考

// ew-splitter 属性 layout: 'horizontal' | 'vertical' // 布局方向，默认 horizontal // ew-splitter-pane 属性 size: string | number // 面板尺寸，支持百分比或像素 min: string | number // 最小尺寸 max: string | number // 最大尺寸 resizable: boolean // 是否可调整大小，默认 true collapsible: boolean // 是否可折叠，默认 false // 事件 resize-start: 拖拽开始 resize: 拖拽中 resize-end: 拖拽结束 collapse: 折叠/展开
